Based Mod

Step by Step Installation Instructions

Step 1:

Download the Zip File from the Main Page

Step 2:

Save the file somewhere you can access it later.

Step 3:

Open Pokemon Online and access Team Builder & Open the Mods Tab.

Step 4:

In the Mods Tab select Install New Mod

Step 5:

Find & Load the zip file you downloaded before.

Step 6:

Finally and most fun! Create your Team and start battling!
